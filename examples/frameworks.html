# Mocha

This tool can either consume the default json (opens new window) reporter or the jUnit (opens new window) reporter for publishing test results.

# JSON Reporter

Run tests by updating the default reporter to json.

mocha tests --reporter json
1

Sample report.json file

{
  "targets": [
    {
      "name": "slack",
      "inputs": {
        "url": "<incoming-webhook-url>"
      }
    }
  ],
  "results": [
    {
      "type": "mocha",
      "files": ["path/to/result.json"]
    }
  ]
}

# jUnit Reporter

Run tests by installing the junit reporter by updating the default reporter to junit.

npm install mocha-junit-reporter --save-dev
mocha test --reporter mocha-junit-reporter
1
