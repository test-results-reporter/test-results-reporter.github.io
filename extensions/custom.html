# Custom

Attach custom extensions.

# Inputs

# > load (string | Function)

  • it could be the path of the file to load which exports a run function.
  • it could be a function

# Examples

# file path

Config file.





 
 
 
 
 
 
 


















{
  "reports": [
    {
      "targets": [
        {
          "name": "teams",
          "inputs": {
            "url": "<some-url>"
          },
          "extensions": [
            {
              "name": "custom",
              "inputs": {
                "load": "./custom.js"
              }
            }
          ]
        }
      ],
      "results": [
        {
          "type": "testng",
          "files": ["path/to/testng-results.xml"]
        }
      ]
    }
  ]
}
